## Wiring Diagram (Pin Mapping)
| Arduino Pin | Connected To            |
|-------------|-------------------------|
| D2, D3, D5  | L298N Motor Driver      |
| D9          | Steering Servo          |
| D10         | Scanning Servo          |
| D11 (Trig)  | Ultrasonic Sensor       |
| D12 (Echo)  | Ultrasonic Sensor       |
| D4, D6, D7, D8, D13 | TCS3200 Color Sensor |
